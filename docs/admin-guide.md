# Nabbis Collections - Admin Guide

## For Non-Technical Staff

### Dashboard
- **Nabbis Dashboard Widget** — Shows today's orders, monthly revenue, product count, customer count, and low stock alerts
- **Nabbis Menu** — Under the WordPress admin sidebar (palm tree icon)

### Homepage Builder
1. Go to **Nabbis → Homepage**
2. Edit Hero Section (heading, subheading, CTA, image/video)
3. Add/edit Featured Collections
4. Add/edit Trust Badges
5. Edit Brand Story section
6. Add/edit Testimonials
7. Edit Newsletter section

### Managing Banners
1. Go to **Nabbis → Banners**
2. Add promotional banners (title, description, image, link)
3. Banners can be placed in slider or promotional sections

### Theme Settings
1. Go to **Nabbis → Theme Settings**
2. Upload logo (light/dark versions)
3. Set favicon
4. Configure delivery info bar text
5. Enable/disable flash sale banner

### Contact & Social
1. Go to **Nabbis → Contact & Social**
2. Set phone, WhatsApp, email, and address
3. Add social media links
4. Add Google Maps embed URL

### Products
- Standard WooCommerce product management under **Products**
- Add variations for size/color combinations
- Set stock levels per variation
- Add product categories, tags, and attributes

### Orders
- View/process orders under **WooCommerce → Orders**
- Add delivery tracking info in the Delivery Details metabox
- View M-Pesa payment details

### M-Pesa Transactions
- View transaction logs under **WooCommerce → M-Pesa Logs**
- Filter by status (pending, completed, failed, timeout)
- Link to associated orders

### Delivery Zones
- Manage under **WooCommerce → Delivery Zones**
- Add/edit/delete zones
- Set rates, free threshold, and ETA per zone

### Customer Features
- Customers can manage wishlist, compare products, view loyalty points, and referrals from their My Account page
- Points awarded automatically on completed orders
- Referral links generated automatically
